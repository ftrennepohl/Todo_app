# Todo_app

A todo SPA built with the purpose of learning PHP, Laravel and React. 

## How to run

Install Laravel

```
/bin/bash -c "$(curl -fsSL https://php.new/install/linux/8.4)"
```

Run

```
git clone https://github.com/ftrennepohl/Todo_app
cd Todo_app
npm install && npm run build
composer run dev
```
